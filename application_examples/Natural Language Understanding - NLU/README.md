# Natural Language Understanding - NLU
## Overview
This repository hosts the supporting content for **Renesas Ready Solutions** in **Artificial Intelligence of Things** (*#aiot*) on **Renesas Microcontrollers**.

##Description
The easy-to-use Renesas hardware platform for Voice User Interface (VUI) solutions is based on the high-performance RA family of 32-bit MCUs and is used for all VUI partner-enabled solutions within the Renesas Ready Partner Network. Each kit demonstrates how easy it is to control your system with a simple voice command interface without extensive coding experience or in-house expertise 
See below for additional details:
* RA8M1 Voice User Demonstration Kit: https://www.renesas.com/en/design-resources/boards-kits/vk-ra8m1
